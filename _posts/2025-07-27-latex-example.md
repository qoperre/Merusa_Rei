$E = mc^2$ 는 유명한 에너지-질량 등가식입니다.

대표적인 정적분 공식:

$$
\int_0^1 x^n \, dx = \frac{1}{n+1}, \quad n \ne -1
$$

부정적분 예시:

$$
\int \sin x \, dx = -\cos x + C
$$

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
